# PlayersDAO
Desafio DIO - Criando uma DAO

# Missão
Nossa missão é devolver o protagonismo ao músico independente, utilizando a governança descentralizada para criar um ecossistema justo, sustentável e colaborativo. Existimos para garantir que quem faz a trilha sonora da noite tenha voz nas decisões, transparência financeira e o apoio de uma rede que valoriza o trabalho artístico real.

# Entrega
A PlayersDAO entrega uma infraestrutura de suporte completo para a carreira do músico independente, incluindo:
 * Pagamentos Instantâneos: Liquidação de cachês via contratos inteligentes, reduzindo atrasos e calotes.
 * Fundo de Reserva Coletivo: Acesso a um tesouro comunitário para manutenção de equipamentos e auxílio em períodos de baixa temporada.
 * Rede de Oportunidades: Curadoria coletiva de bares e eventos que respeitam o artista.
 * Poder de Voto: Participação direta na definição de taxas, parcerias e investimentos da organização.

# Roadmap
 * Fase 1 (Comunidade): Criação do canal oficial no Discord para networking e troca de agendas.
 * Fase 2 (Governança): Lançamento do token de reputação para votações coletivas.
 * Fase 3 (Financeiro): Implementação de pagamentos via contratos inteligentes e fundo mútuo.
